Aciona a função jogar para gerar um número aleatoriamente

Depois Verifica qual é nível o jogador escolheu e define o valor do numero aleatório.

O jogador tenta adivinhar o valor com base na quantidade de chances que o nível permite.

Analisa e exibe se o palpite do jogador é igual, menor ou maior que o numero_secreto, mostrando o resultado.

Se o jogador vencer, a mensagem é "venceu", mas se estiver jogando no modo "extra", a mensagem muda para "Sao feras!".
